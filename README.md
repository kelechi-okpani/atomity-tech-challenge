
# Cloud Efficiency Explorer

A high-fidelity, interactive dashboard built to visualize cloud infrastructure costs and resource usage. This project demonstrates advanced React patterns, fluid animations, and a deep drill-down navigation system.


## ✨ Features
* **Interactive Drill-down:** Navigate from Clusters to Namespaces, Pods, and finally specific Resource Types.
* **Fluid Animations:** Smooth layout transitions using Framer Motion (including shared layout animations).
* **Smart Caching:** Real-time data fetching with TanStack Query to ensure instant UI responses.
* **Responsive Design:** Fully adaptive layout using CSS Container Queries.


## 🚀 Tech Stack
* **Framework:** Next.js (App Router)
* **Styling:** Tailwind CSS
* **Animations:** Framer Motion
* **Data Fetching:** TanStack Query (React Query)
* **Icons:** Lucide React


## 🛠️ Setup
1. Clone the repository.
2. Install dependencies: `npm install`
3. Run the development server: `npm run dev`
4. Open `http://localhost:3000` to view.
